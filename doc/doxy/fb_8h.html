int fb_init ( VideoFb *  fb)

Initialize the framebuffer instance.

Initializes all private members in the given instance and checks if the framebuffer is valid. If successful, the framebuffer memory will be cleared with the color black. This is not necessary, since VBE normally clears the framebuffer when changing modes, but does not hurt.

Returns
int E_SUCCESS Framebuffer initialized successfully E_ARGNULL fb argument was NULL

Calculates a pixel offset in a framebuffer at x and y.

Calculates the offset in the framebuffer for a pixel at coordinates x and y. The calculated offset will be set in the offset pointer. This offset is the number of bytes from the framebuffer location and cannot be used as an index. The x and y coordinates should be within the bounds of the framebuffer's resolution, otherwise an offset that is outside the bounds of the linear framebuffer array will be returned. Note that the bounds are not checked due to performance reasons.

Returns
int E_SUCCESS: The offset was calculated and stored in the offset pointer E_ARGNULL: fb or offset was set to NULL
Parameters
fbPointer to framebuffer to calculate an offset from struct's location member (

VideoFb)
Parameters
offsetPointer to store the calculated offset in.
xThe X coordinate, should be 0 <= x < fb->width
yThe Y coordinate, should be 0 <= y < fb->height
