Intel Network driver. Contains structures and public methods to perform network functions. Intel Shared Memory Architecture

CU := Command Unit CB := Command Block CBL := Command Block List TCB := Transmit Command Block TBD := Transmit Buffer Descriptor IA := Individual Address

RU := Receive Unit RFD := Receive Frame Descriptor RFA := Receive Frame Area

ARP := Address Resolution Protocol
