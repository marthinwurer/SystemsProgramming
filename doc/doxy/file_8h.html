Gets the child of a directory by index; supports sorting.

FILEHANDLE file must be a valid handle returned by IoFileOpen
Parameters

plength[in] defines size of output buffer in bytes. [out] number of bytes in answer
out[out] string path to file; null if plength is less than the size of the path.
