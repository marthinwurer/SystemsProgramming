int vcon_redraw ( VConCtrl *  ctrl)

Writes the current virtual console buffer to the screen.

For VCON_MODE_VGATEXT, the buffer is copied to memory location 0xB8000 For VCON_MODE_GRAPHICS, the buffer is drawn using the associated paint context.
