VideoTiming *  timing

Parses the EDIDRecord and tries to find the native timing.

Reads the given EDIDRecord and stores (if found) the native or preferred timing in the given timing pointer. If the native timing cannot be found, EDID_NOTIMING is returned and the timing struct is not modified. Otherwise, the timing struct is set accordingly and EDID_SUCCESS is returned.
